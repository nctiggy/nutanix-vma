## Codebase Patterns

- Hand-write type files in api/v1alpha1/ (do NOT use `kubebuilder create api` -- it generates controller stubs)
- After any *_types.go change: `make generate && make manifests`
- Register new API groups in cmd/main.go init() with utilruntime.Must(pkg.AddToScheme(scheme))
- corev1.PersistentVolumeMode and corev1.PersistentVolumeAccessMode work with kubebuilder Enum validation
- kubebuilder printcolumn markers provide `kubectl get` table output
