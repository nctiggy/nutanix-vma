## US-010: Validation Engine

### Status: Implemented, pending CI verification

### What was done:
- Created internal/validation/validator.go: Validate(ctx, vm, opts) returns []Concern with 13 validation rules
- Source-side rules (always run, no K8s client needed):
  - Unmapped storage container (Error): checks each disk's StorageContainerRef against StorageMap by ID or name, deduplicates by container
  - Unmapped subnet (Error): checks each NIC's NetworkRef against NetworkMap by ID or name, deduplicates by subnet
  - GPU present (Warning): flags VMs with GPU attachments requiring manual target configuration
  - Volume Group disk (Error): flags disks with nil BackingInfo or nil StorageContainerRef (not CDROM)
  - NETWORK_FUNCTION_NIC (Error): flags unmigrable network function NICs
  - DIRECT_NIC (Warning): flags passthrough NICs requiring SR-IOV on target
  - MAC conflict (Warning): checks NIC MACs against ExistingMACs map (lowercase comparison)
  - IDE disk (Info): flags IDE bus disks that will be mapped to SATA on KubeVirt
- Target-side rules (skip gracefully when client.Reader is nil):
  - Namespace missing (Error): checks target namespace exists via client.Get
